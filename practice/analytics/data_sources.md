## 6. JEITA 民生用電子機器国内出荷統計

テレビ・PC・スマートフォン等の出荷台数・金額（メーカー側データ）。

- **統計ページ:** https://www.jeita.or.jp/japanese/stat/shipment/index.htm
- ※ Excel（.xls）形式。要「出典：JEITA」明記。

## 7. JEMA 民生用電気機器 国内出荷実績

冷蔵庫・洗濯機・エアコン等の白物家電出荷統計。

- **統計ページ:** https://www.jema-net.or.jp/stat/kakoku.html

## 分野別 推奨データセット まとめ

| 調査したい分野 | 推奨データ |
|-------------|----------|
| エンターテイメント全般（映画・音楽・ゲーム） | 家計調査「教養娯楽」+ 特定サービス産業動態統計 |
| 家電（テレビ・PC等） | 家計調査「教養娯楽用耐久財」+ JEITA 出荷統計 + 消費動向調査（普及率） |
| 白物家電（冷蔵庫・エアコン等） | 家計調査「家具・家事用品」+ JEMA 出荷統計 |
| デジタル・サブスク消費 | 家計消費状況調査（ICT支出） |
| 消費者マインド | 消費動向調査（消費者態度指数） |
