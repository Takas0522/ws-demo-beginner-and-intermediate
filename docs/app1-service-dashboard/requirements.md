# App1: サービスダッシュボード 機能要件

## 1. 概要

複合企業が提供する各電子サービスの利用状況・KPI情報をリアルタイムかつ多角的に可視化するダッシュボードアプリケーションです。事業部単位での売上・コスト管理から、サービスごとの詳細なユーザー行動分析・ABテスト結果まで一元管理します。

---

## 2. ドメインモデル概要

```
会社
└── 事業部 (BusinessUnit)
    └── サービス (Service)
        ├── サービスプラン (ServicePlan)
        ├── 利用契約 (Subscription)
        │   └── ユーザー (User)
        ├── 売上明細 (Revenue)
        ├── 原価明細 (Cost)
        └── ABテスト (ABTest)
            └── バリアント (Variant)
                └── 測定結果 (ABTestResult)
```

---

## 3. 機能要件

### 3.1 ダッシュボード表示

#### 3.1.1 全社サマリービュー

| 表示項目 | 説明 |
|----------|------|
| 全サービス合計利用ユーザー数 | アクティブユーザー数（MAU/DAU） |
| 全社売上合計 | 当月・当四半期・当年度累計 |
| 全社粗利合計 | 売上 - 原価 |
| 全社粗利率 | 粗利 / 売上 × 100 |
| 事業部別売上比率 | 円グラフまたは棒グラフ |
| サービス数 | 提供中サービス総数 |

#### 3.1.2 事業部別ビュー

| 表示項目 | 説明 |
|----------|------|
| 事業部名 | - |
| 担当サービス一覧 | 各サービスへのドリルダウンリンク付き |
| 事業部売上合計 | 当月・累計 |
| 事業部原価合計 | 当月・累計 |
| 事業部粗利・粗利率 | - |
| アクティブユーザー数合計 | 事業部内サービス合算 |

#### 3.1.3 サービス別詳細ビュー

| 表示項目 | 説明 |
|----------|------|
| サービス名・概要 | カテゴリ・提供開始日など |
| MAU / DAU | 月次・日次アクティブユーザー数 |
| 新規登録ユーザー数 | 期間別（日/週/月） |
| 解約ユーザー数・解約率 | チャーン分析 |
| 売上（プラン別内訳） | 有料プラン別・月次推移グラフ |
| 原価内訳 | インフラ費・ライセンス費・人件費など |
| 粗利・粗利率 | - |
| ARPU | Average Revenue Per User |
| LTV | Life Time Value（平均値） |
| NPS スコア | 任意取得時 |

#### 3.1.4 ABテスト結果ビュー

| 表示項目 | 説明 |
|----------|------|
| テスト名・対象サービス | - |
| 実施期間 | 開始日〜終了日（進行中/終了ステータス） |
| バリアント一覧 | Control / Treatment A / Treatment B など |
| バリアントごとの割り当てユーザー数 | - |
| 主要指標（Primary Metric） | コンバージョン率・クリック率・売上等 |
| 統計的有意性 | p値・信頼区間 |
| 勝者バリアント | 有意差がある場合に表示 |
| 二次指標（Secondary Metrics） | 副作用確認用の補助指標 |

---

### 3.2 フィルタリング・検索

| 機能 | 詳細 |
|------|------|
| 期間絞り込み | 日・週・月・四半期・年・カスタム期間 |
| 事業部絞り込み | マルチセレクト |
| サービスカテゴリ絞り込み | マルチセレクト |
| サービス検索 | サービス名・IDによるインクリメンタル検索 |
| ABテスト絞り込み | 実施中・終了済・全件 |

---

### 3.3 CSV エクスポート

#### エクスポート対象

| エクスポート種別 | 含まれる主なカラム |
|-----------------|-------------------|
| 全社KPIサマリー | 日付, 事業部ID/名, サービスID/名, MAU, DAU, 新規ユーザー数, 解約数, 売上, 原価, 粗利, 粗利率, ARPU, LTV |
| サービス詳細 | 上記 + プラン別売上, 原価内訳, NPS |
| ABテスト結果 | テストID/名, サービスID/名, 開始日, 終了日, ステータス, バリアント名, 割り当てユーザー数, プライマリ指標値, p値, 信頼区間下限・上限, 勝者フラグ |
| ユーザー集計データ | 日付, サービスID/名, ユーザーセグメント, MAU, DAU, 新規, 解約 |

#### エクスポート要件

- 文字コード: UTF-8 with BOM（Excel対応）
- 日付フォーマット: YYYY-MM-DD
- 数値フォーマット: カンマなし（Excel側で整形）
- ダウンロード方式: ブラウザダウンロード（ファイル名に期間・種別を含む）

---

### 3.4 グラフ・可視化

| グラフ種別 | 用途 |
|-----------|------|
| 折れ線グラフ | ユーザー数推移・売上推移 |
| 棒グラフ | 事業部別・サービス別比較 |
| 積み上げ棒グラフ | プラン別売上内訳推移 |
| 円グラフ / ドーナツグラフ | 事業部別売上比率 |
| ファネルチャート | 登録〜課金コンバージョン |
| ABテスト比較バー | バリアント別指標比較 |

## 4. 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンス | ダッシュボード初期表示 3秒以内 |
| データ更新頻度 | 日次バッチ更新（リアルタイム更新は将来対応） |
| 対応ブラウザ | Chrome / Edge 最新版 |
| 認証 | ログイン認証必須（社内利用想定） |
| 権限管理 | 事業部スコープによる閲覧制限（将来対応） |

## 5. APIエンドポイント概要（Backend）

| メソッド | パス | 説明 |
|----------|------|------|
| GET | /api/summary | 全社KPIサマリー取得 |
| GET | /api/business-units | 事業部一覧取得 |
| GET | /api/business-units/{id}/summary | 事業部別KPIサマリー取得 |
| GET | /api/services | サービス一覧取得（絞り込み対応） |
| GET | /api/services/{id} | サービス詳細KPI取得 |
| GET | /api/services/{id}/ab-tests | サービスのABテスト一覧取得 |
| GET | /api/ab-tests/{id} | ABテスト詳細結果取得 |
| GET | /api/export/kpi-summary | KPIサマリーCSVエクスポート |
| GET | /api/export/service-detail | サービス詳細CSVエクスポート |
| GET | /api/export/ab-tests | ABテスト結果CSVエクスポート |
| GET | /api/export/user-metrics | ユーザー集計CSVエクスポート |

クエリパラメータ共通: `from`, `to`（日付範囲）, `businessUnitIds`, `serviceIds`, `categoryIds`

## 6. 起動情報

| レイヤー | ポート | 備考 |
|----------|--------|------|
| Frontend (React) | 3001 | Vite dev server / 本番はNginx等 |
| Backend (ASP.NET Core) | 5001 | HTTP |
| Database (PostgreSQL) | 5433 | ホスト側ポート（コンテナ内5432） |

個別起動コマンド:

```bash
# Frontend
cd target-app/app1-service-dashboard/frontend && npm run dev

# Backend
cd target-app/app1-service-dashboard/backend && dotnet run

# Database (Docker)
docker compose -f target-app/app1-service-dashboard/database/docker-compose.yml up -d
```
